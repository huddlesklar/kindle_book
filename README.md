### 转 azw3

有时, 通过官方工具转成`mobi`之后, 格式就乱了. 这时可以转成`azw3`格式查看, kindle 也是支持的. 这就简单多了, 很多在线工具网站都支持. 

比如: [https://epub2kindle.com/zh/](https://epub2kindle.com/zh/)
